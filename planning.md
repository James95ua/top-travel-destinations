# Planning Documentation

## Website Title
Top Travel Destinations

## Website Purpose
The purpose of this website is to introduce users to some of the most popular travel destinations in the world. The website provides useful information about attractions, travel tips, prices, and the best seasons to visit. It was designed to be simple, clear, and easy to use.

## Target Audience
The target audience for this website is people who are interested in travel and want simple travel information before planning a holiday. This includes students, young adults, families, and general users who want ideas for destinations.

## Website Objectives
The main objectives of the website are:
- to introduce popular destinations
- to provide useful travel advice
- to compare prices and seasons
- to help users plan a trip more easily
- to present information in a clear and attractive way

## How the Site Serves the Audience
The site serves its audience by giving basic travel information in a beginner-friendly format. The navigation is simple, the content is easy to read, and the pages are connected in a logical structure.

## Design Principles

### Contrast
Contrast was used through colour choices, heading sizes, and section backgrounds. Dark blue headings and navigation were placed against a light background to create clear visual difference. This makes the content easier to read.

### Repetition
Repetition was used by keeping the same navigation menu, colour palette, font style, card design, and footer on every page. This creates consistency across the website.

### Alignment
Alignment was used by arranging headings, paragraphs, cards, tables, and form fields in a neat structure. This helps the website look organised and professional.

### Proximity
Proximity was used by grouping related content together. For example, destination descriptions are grouped with their images and lists, and travel tips are grouped into clear sections.

## Site Structure
The website has five pages:

- Home
- Destinations
- Travel Tips
- Prices & Seasons
- Contact

## Site Map
Home  
├── Destinations

├── Prices & Seasons  
└── Contact

## Navigation Structure
A top navigation bar was used on every page. This makes it easy for users to move between sections of the website. The same navigation appears on all pages to maintain consistency and improve usability.

## User Journey
Users usually begin on the Home page, where they are introduced to the website. From there they can move to the Destinations page to explore places, the Travel Tips page for useful advice, the Prices & Seasons page for comparisons, and the Contact page to enter travel details.

## Visual Design

### Wireframes
Wireframes were created for the Home page, Destinations page, and Contact page. These wireframes helped plan the layout before building the website.

### Colour Scheme
The website uses a simple travel-themed colour palette:
- Dark blue for navigation and headings
- Light blue for secondary highlights
- Gold accent colour for hover effects
- White and light grey backgrounds for readability

These colours were chosen to create a calm and professional travel style.

### Typography
A simple and readable font style was used. Headings are larger and bold to create visual hierarchy, while body text is clear and easy to read. This improves accessibility and readability.

### Layout Approach
The homepage uses CSS Grid for featured destination cards. The destinations page uses Flexbox for card layout. The structure is responsive so that it works on both large and small screens.

## Development Tools
The following tools were used in this project:
- Visual Studio Code for writing and editing code
- GitHub for storing the project files
- GitHub Pages for publishing the website online
- Web browser tools for testing and checking responsiveness

## Code Generators and Frameworks Considered
No frameworks were used in this project because the aim was to develop the website using HTML5 and CSS3. This helped demonstrate understanding of core web authoring skills.

## Conclusion
This planning process helped organise the website before development. It ensured that the final website had a clear purpose, audience, structure, and design style.
