# Top Travel Destinations

## Overview
Top Travel Destinations is a multi-page website created as part of the Web Authoring module. The purpose of the website is to introduce users to popular travel destinations and provide useful information about attractions, travel tips, prices, and the best seasons to visit.

## Website Purpose
The website was designed to help users explore travel ideas in a simple and clear way. It is aimed at people who are interested in travel and want beginner-friendly information before planning a holiday.

## Pages Included
- **Home page (`index.html`)**  
  Introduces the website and presents featured destinations.

- **Destinations page (`destinations.html`)**  
  Provides details about Paris, Rome, Tokyo, and Dubai.

- **Travel Tips page (`tips.html`)**  
  Gives useful travel advice, including planning, packing, safety, and saving money.

- **Prices & Seasons page (`prices.html`)**  
  Includes tables comparing travel costs and the best seasons to visit each destination.

- **Contact page (`contact.html`)**  
  Contains a form that allows users to enter travel preferences and send a message.

## Features
- Five interconnected HTML pages
- External CSS stylesheet
- Responsive design
- CSS Flexbox layout
- CSS Grid layout
- Images with alt text
- Ordered and unordered lists
- Two data tables
- Contact form with multiple input types
- Internal and external links
- Semantic HTML5 structure
